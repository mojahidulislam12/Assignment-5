1.What is the difference between var, let, and const?
Ans: In JavaScript, var, let, and const are used to declare variables, but they behave differently in terms of scope, re-declaration, and reassignment.
Var:
-Used in older JavaScript (before ES6).
-It is function scoped, not block scoped.
-Can be re-declared and reassigned.
Let:
-Introduced in ES6 (2015).
-Block scoped (works inside { }).
-Cannot be re-declared, but can be reassigned.
Const:
-Also introduced in ES6.
-Block scoped like let.
-Cannot be reassigned or redeclared.
-Must be initialized when declared.

2. What is the spread operator (...)?
Ans: The spread operator (...) is used to expand elements of an array or object into individual elements, commonly used for copying, merging, and passing values in functions.

3.What is the difference between map(), filter(), and forEach()?
Ans: In JavaScript, map(), filter(), and forEach() are array methods used to iterate over arrays, but they work differently and return different results.
map():
-Used to transform each element of an array.
-Returns a new array with the same length.
filter():
-Used to select elements that match a condition.
-Returns a new array with only the filtered elements.
ForEach():
-Used to run a function for each array element.
-Does not return a new array.

4.What is an arrow function?
Ans:An Arrow Function is a shorter way to write a function in JavaScript, introduced in ES6 (ECMAScript 2015). It uses the => (arrow) syntax.

5.What are template literals?
Ans: Template Literals are a feature in JavaScript (ES6) that allow you to create strings more easily using backticks (``) instead of single ' ' or double " " quotes.
